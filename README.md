et-tinypng
==========

A python script to shrink picture by [tinypng.com](http://tinypng.com)

## Config

you need the modify the __etiny.cfg__ file.

```cfg
[etiny]

key         = your tinypng key
outputdir   = ouput dir name 
inputdir    = input dir name
```

## Run

run the command as follow:

```cmd
python tinypng.py
```
